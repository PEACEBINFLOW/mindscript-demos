# 3.1-basic-names-demo.md  
## Demo: Simple Name Generator with Runtime

Goal:
- Show MindScript Runtime executing a simple staged prompt.

### Steps

1. Create a file: `demos/simple-names.ms.md` inside `mindscript-runtime`.

Example content:

```markdown
<MIND_SCRIPT>
<STAGE_1>
You are an assistant generating human names.

Task:
- Generate 10 unique human names.
- Output as a bullet list.
</STAGE_1>
</MIND_SCRIPT>
