## Getting Started with MindScript Demos

To run the demos, you should have:

- `mindscript-core` (spec / mental model)
- `mindscript-templates` (ready-made templates)
- `mindscript-ledger` (for logging)
- `mindscript-runtime` (to execute programs)
- `mindscript-search` (to recall / analyze)
